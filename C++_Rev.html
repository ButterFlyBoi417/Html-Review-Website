C++ Review

Hello World:

  1. C++ is a general-purpose coding language.
  2. C++ runs line by line, from top to bottom.
  3. std::cout is how you output to the terminal:
    • std::cout << "Good luck!\n";

Compile & Execute:

  1. Compilation and execution using the terminal.
    • g++ hello.cpp -o hello
    • ./hello
  2. Single line comments can be created using //.
  3. Multi-line comments can be created using /* */.

Variables:

  1. A variable represents a particular piece of your computer’s memory that has been set aside for you to use to store, retrieve, and manipulate data.
  2. C++ basic data types include:
    • int: integers
    • double: floating-point numbers
    • char: individual characters
    • string: sequence of characters
    • bool: true/false
  3. Single equal sign = indicates assignment, not equality in the mathematical sense.
  4. cin is how to receive input from the user.

Conditionals & Logic:

  1. An if statement checks a condition and will execute a task if that condition evaluates to true.
  2. if / else statements make binary decisions and execute different code blocks based on a provided condition.
  3. We can add more conditions using else if statements.
  4. Relational operators, including <, >, <=, >=, ==, and != can compare two values.
  5. A switch statement can be used to simplify the process of writing multiple else if statements. The break keyword stops the remaining cases from being checked and executed in a switch statement.
  6. If you add #include <cstdlib>, you can use the std::rand() function to generate a random number.
  7. If you add % 10 to the end of a random number generator it will generate a number 0-9. This works with more numbers than the example used here.

Logical Operators:

  1. &&: the and logical operator
  2. ||: the or logical operator
  3. !: the not logical operator
  4. When A / B = Q, where Q is the quocient, what is the remainder(R)? When A % B = R, you get the remainder in place of the quocient.

Loops:

  1. Loops perform repetitive actions so we don’t have to code those actions manually every time.
  2. How to write while loops with a continue condition.
  3. How to write for loops with a counter that increments or decrements.

Errors:

  1. Compile-time errors: Errors found by the compiler.
  2. Link-time errors: Errors found by the linker when it is trying to combine object files into an executable program.
  3. Run-time errors: Errors found by checks in a running program.
  4. Logic errors: Errors found by the programmer looking for the causes of erroneous results.

Vectors:

  1. Vectors are a sequence of elements that you can access by an index.
    • std::vector<int> even = {2, 4, 6, 8, 10};
  2. The first index in a vector is 0.
  3. Some of the functions that come with vectors:
    • .push_back()
    • .pop_back()
    • .size()
  4. We can use a for loop to iterate through a vector.

Functions:

  1. A function is a named group of statements that do something together.
  2. Functions allow you to create more flexible, modular, and DRY code.
  3. C++ has many built-in functions that you can use.
  4. Functions are called like function_name();
  5. A function has a declaration with a return type and possible parameters.
  6. A function has a definition (or body) with a group of statements and a possible return value.
  7. void functions do not have return values.
  8. Functions with a return value have return statements.
  9. Parameters are variables used as placeholders for function input values.
  10. Arguments are a function’s actual input values.
  11. pow(a, b) is a function that will raise a to the power of b

Functions: Scope & Flexibility:

  1. Scope is the region of code that has access to an element.
    • Globally scoped variables are accessible everywhere.
    • A variable created inside a function has local scope and can’t be accessed outside the function.
  2. C++ functions are usually split to make code more modular:
    • The declaration in a header file.
    • The definition in another .cpp file.
  3. Programs with multiple .cpp files need to be linked at compile time:
    • g++ main.cpp fns.cpp
  4. Header files must be included in the file with main():
    • #include "fns.hpp"
  5. You can define a function inline using the inline keyword, which may or may not improve execution speed.
  6. Default arguments can be added to function declarations so that you can call the function without including those arguments.
  7. You can overload C++ functions so that they handle different types of input and return different types.
  8. A function template enables a function to behave the same with different types of parameters.

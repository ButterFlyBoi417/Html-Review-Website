Method Calls and Input:

  1. Call a method with its name and parentheses:
    • VisitPlanets();
  2. Store a method’s returned value in a variable:
    • double result = Math.Round(3.14159, 2);
  3. Define a basic method with the following syntax:
    • static void VisitPlanets() { }
  4. Every time an application is started, the Main() method is called.
  5. Values passed to a method are called arguments. When defined in the method, they are parameters.
  6. Method parameters can only be used within the method body.
  7. Method parameters can be optional if given a default value using equals = syntax:
    • static void VisitPlanets(int numberOfPlanets = 0)
  8. When calling a method, pass arguments by position or by name. If using names, use the colon (:) syntax:
    • VisitPlanets(numberOfPlanets: 9);
  9. In method overloading, multiple methods can have the same name, as long as they have different method signatures.
  10. A method signature is a method’s name and parameter types in order.
